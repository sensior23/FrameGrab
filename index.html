How to Use
Capture frames in 3 steps
📱 Mobile mode: Screen share isn't supported on mobile browsers. FrameGrab automatically uses YouTube's CDN thumbnails — timestamps work great! For Snap Region or Full Frame live capture, use a desktop browser.
1

Paste YouTube URL

Paste any YouTube link and tap Load →.

2

Set Timestamps

Enter times like 1:23 or 1:23:45. Load chapters from the video or bulk-import.

3

Extract or Snap

Snap Region for live area selection, Full Frame for the whole video, or Extract All for all timestamps at once.

Extract All to grab CDN screenshots for all timestamps. Tap 📸 Capture Current Frame to grab whatever timestamp is showing in the video.

4

✨ Magic Eraser

After capturing, tap ✨ on any screenshot. Paint over watermarks, logos, or anything else. Tap Save when done.

🔑 AI Notes: Add an Anthropic API key (tap the badge in the nav) to get AI-generated notes for each screenshot.
