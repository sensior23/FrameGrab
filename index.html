How to Use FrameGrab
Capture frames & notes in 3 steps
1

Paste YouTube URL

Paste any YouTube link and click Load Video →.

2

Set timestamps

Use H:M:S boxes. Load chapters with 📑 or import bulk with ⬆.

3

Snap or Extract

Snap Region for a selected area, Full Frame for the whole video. All UI hides before capture — no controls appear in screenshots. Use the ⊞ Controls sidebar button for fullscreen use.

4

✨ Magic Eraser

After capturing, click the ✨ button on any screenshot. Paint over YouTube watermarks, logos, or anything else to erase. Click Save & Replace when done.

📸 Clean Screenshots: The UI hides before every capture so only the video frame is visible. No controls, no buttons. If any UI elements appear, use ✨ Magic Eraser to remove them.
